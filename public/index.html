Holidaypirates fake api server

https://github.com/holidaypirates/fake-api

Endpoints

Get a hotels list

GET https://fake-hotel-api.herokuapp.com/api/hotels

Request Parameters

QUERY PARAMETER VALUE
no_error Optional. Prevents server from errors if set.

By default server returns error sometimes.

force_error Optional. Enforces server error if set.
count Optional. The number of hotel objects to return. Default: random count from 0 to 500.
min_stars Optional. Minimum hotel stars.
max_price Optional. Maximum offer price.

Response example

curl https://fake-hotel-api.herokuapp.com/api/hotels

Get reviews

GET https://fake-hotel-api.herokuapp.com/api/reviews

Request Parameters

QUERY PARAMETER VALUE
hotel_id Required. Hotel id.

Response example

curl https://fake-hotel-api.herokuapp.com/api/reviews?hotel_id=13ef1108-7f18-40c7-ac0f-0e743b015755

[
  {
    "name": "Nigel Kub", //commenter name
    "comment": "Rerum est suscipit adipisci odio hic.", //comment
    "positive": true, //false if comment is negative
    "hotel_id": "13ef1108-7f18-40c7-ac0f-0e743b015755" //hotel id
  },
  ...
]
